---

Запуск локального сервера для разработки

```
npm start
```


Сборка проекта

```
npm run build
```


Публикация проекта (перед выполнением нужно собрать проект)

```
npm run gh-pages
```


Все разработка ведётся в директории `source/`.**

Итоговый код попадает в директорию `build/`.**
